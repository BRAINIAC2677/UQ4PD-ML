Data:
1. all_file_user_metadata.csv -- contains demographic information for all participants who recorded at least one task.
2. all_task_ids.txt -- contains the unique ids of 845 participants completing all three tasks
3. demographic_details.csv -- demographic information for 845 participants completing all three tasks
4. dev_set_participants.txt -- unique ids of participants randomly selected for the validation set
5. test_set_participants.txt -- unique ids of participants randomly selected for the test set
6. facial_expression_smile/facial_dataset.csv -- extracted features for the facial expression (smile) task
7. finger_tapping/features_demography_diagnosis_Nov22_2023.csv -- extracted features for the finger-tapping task
8. quick_brown_fox/wavlam_fox_features.csv -- extracted features for the speech task

Code:
1. unimodal_models/{task_name}/constants.py -- paths set up to run task-specific model without MC-dropout
2. unimodal_models/{task_name}/constants_baal.py -- paths set up to run task-specific model *with* MC-dropout
3. unimodal_models/{task_name}/unimodal_{task_tag}.py -- code for the tasks-specific model without MC-dropout
4. unimodal_models/{task_name}/unimodal_{task_tag}_baal.py -- code for the tasks-specific model *with* MC-dropout
5. fusion_models/{module_name}/constants.py -- paths set up to run the experiment
6. fusion_models/{module_name}/{experiment_name}.py -- code for the experiment. For example, fusion_models/uncertainty_aware_fusion.py contains the code for the UFNet used in this study.
